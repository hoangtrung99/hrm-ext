## Hướng dẫn sử dụng

1. Tải xuống từ Github `https://github.com/hoangtrung99/hrm-ext`
2. Giải nén
3. Import vào Chrome
  - 3.1. Mở chrome extension `chrome://extensions/`
  - 3.2. Mở chế độ developer mode.
  - 3.3. Load unpacked -> chọn tới folder vừa giải nén.
